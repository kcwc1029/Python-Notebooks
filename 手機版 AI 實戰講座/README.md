### AI(人工智慧)

## 什麼是 AI(人工智慧)？

AI(Artificial Intelligence，人工智慧) 指的是：「讓電腦模擬人類智慧行為的技術。」

例如：看懂圖片、聽懂聲音、回答問題、推薦影片、自動翻譯

- 以前的電腦：你寫什麼規則，它就做什麼。
- 現在的 AI：它可以「從資料中學習」。

### Machine Learning(機器學習)

機器學習是 AI 的其中一種方法。

核心概念：「不要手寫規則，而是讓電腦從資料中自己找規律。」

例如你給 AI：

- 10000 張貓圖片
- 10000 張狗圖片

它會慢慢學會：

- 什麼叫貓。
- 什麼叫狗。

### Deep Learning(深度學習)

深度學習是機器學習的一種。

差別在：它使用「神經網路」。(靈感來自人類大腦)。

像ChatGPT、Midjourney、Gemini，背後幾乎都靠深度學習。

- [什麼是人工智慧、機器學習和深度學習](https://www.youtube.com/watch?v=9sn52iNIUhY)
- [機器學習與深度學習到底差在哪裡?](https://www.youtube.com/watch?v=-mtjPTgYGZM)

### 深度學習為什麼會爆發？

以前：

- 電腦不夠強。
- 資料不夠多。

近幾年：

- GPU 變強
- 網路資料爆炸成長
- 雲端運算成熟

深度學習直接大爆發。

像：ChatGPT、Midjourney、Gemini，背後幾乎都靠深度學習。

## 什麼是 LLM？

LLM = Large Language Model 大型語言模型

它讀過大量：

- 網頁
- 書籍
- 新聞
- 程式碼
- 對話資料

然後學習：「人類文字的規律。」
