# AI 入門教材：從人工智慧到生成式 AI

這幾年你大概已經發現一件事。不管是手機、社群平台、電商、YouTube，甚至銀行客服，背後幾乎都開始出現 AI。

以前大家覺得 AI 很遙遠，像電影裡的機器人。現在不一樣了。

你每天滑手機時：

- TikTok 幫你推薦影片
- Google 幫你自動翻譯
- ChatGPT 幫你寫文章
- Gemini 幫你整理重點
- Canva 幫你生成設計圖

其實都已經是 AI 在工作。

## 什麼是 AI(人工智慧)？

AI(Artificial Intelligence，人工智慧) 指的是：「讓電腦模擬人類智慧行為的技術。」

例如：看懂圖片、聽懂聲音、回答問題、推薦影片、自動翻譯

- 以前的電腦：你寫什麼規則，它就做什麼。
- 現在的 AI：它可以「從資料中學習」。

## AI、機器學習、深度學習差在哪？

因為很多人會把這三個東西混在一起。其實它們像是「包含關係」。

![](./手機版%20AI%20實戰講座_assets/圖解AI、機器學習、深度學習.jpg)

### AI(人工智慧)

- [一圖解說人工智能、機器學習、深度學習之分別](https://www.youtube.com/watch?v=OZkVmsLNKl4)

只要讓電腦看起來像有智慧，都算 AI。

例如：

- 規則式聊天機器人
- 自動導航
- 遊戲 NPC
- 專家系統

### Machine Learning(機器學習)

機器學習是 AI 的其中一種方法。

核心概念：「不要手寫規則，而是讓電腦從資料中自己找規律。」

例如：

你給 AI：

- 10000 張貓圖片
- 10000 張狗圖片

它會慢慢學會：

- 什麼叫貓。
- 什麼叫狗。

機器學習常見應用

- 垃圾郵件判斷
- 股票預測
- 推薦系統
- 信用評分
- 詐騙偵測

### Deep Learning(深度學習)

深度學習是機器學習的一種。

差別在：它使用「神經網路」。(靈感來自人類大腦)。

#### 深度學習為什麼會爆發？

以前：

- 電腦不夠強。
- 資料不夠多。

近幾年：

- GPU 變強
- 網路資料爆炸成長
- 雲端運算成熟

深度學習直接大爆發。

像：ChatGPT、Midjourney、Gemini，背後幾乎都靠深度學習。

## 什麼是 LLM？

- [YouTube 影片 1](https://www.youtube.com/watch?v=ERr6J1bDXUI&list=PL7O5Ubado0Q3297OGDkjALjNK3BHb3VxN&index=2&utm_source=chatgpt.com)
- [YouTube 影片 2](https://www.youtube.com/watch?v=la9nhavGG5M&list=PL5y2P1AqpsZ-eD0uitRVWRybSX0DZdWNG&index=3&utm_source=chatgpt.com)

LLM = Large Language Model 大型語言模型

它讀過大量：

- 網頁
- 書籍
- 新聞
- 程式碼
- 對話資料

然後學習：「人類文字的規律。」

## 什麼是生成式 AI？

- [生成式 AI 影片 1](https://www.youtube.com/watch?v=bn4OpKNcf6A&list=PL7O5Ubado0Q3297OGDkjALjNK3BHb3VxN&utm_source=chatgpt.com)
