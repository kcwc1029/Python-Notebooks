# AI 為什麼這麼強？它到底怎麼「看懂」世界

## 什麼是 AI(人工智慧)？

AI(Artificial Intelligence，人工智慧) 指的是：「讓電腦模擬人類智慧行為的技術。」

只要讓電腦看起來像有智慧，都算 AI。

例如：看懂圖片、聽懂聲音、回答問題、推薦影片、自動翻譯

- 以前的電腦：你寫什麼規則，它就做什麼。
- 現在的 AI：它可以「從資料中學習」。

### Machine Learning(機器學習)

機器學習是 AI 的其中一種方法。

核心概念：「不要手寫規則，而是讓電腦從資料中自己找規律。」

例如你給 AI：

- 10000 張貓圖片
- 10000 張狗圖片

它會慢慢學會：

- 什麼叫貓。
- 什麼叫狗。

### Deep Learning(深度學習)

深度學習是機器學習的一種。

差別在：它使用「神經網路」。(靈感來自人類大腦)。

像ChatGPT、Midjourney、Gemini，背後幾乎都靠深度學習。
